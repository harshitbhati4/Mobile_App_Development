## 🛠️ Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/harshitbhati4/Mobile_App_Development.git
   cd Mobile_App_Development/Experiments
   ```

2. **Open any experiment in Android Studio or VS Code**

3. **Run the app**
   ```bash
   flutter pub get
   flutter run
   ```
