## 🛠️ Getting Started

```bash
git clone https://github.com/harshitbhati4/Mobile_App_Development.git
cd Mobile_App_Development/Experiments
flutter pub get
flutter run
```

✅ Run each experiment individually by opening its folder in your preferred IDE.
